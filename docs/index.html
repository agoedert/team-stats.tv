How it works

The problem

Nowadays, teams rely on many tools to check their work along all stages of software creation. Tools like CI servers, server and code monitoring, statistics about your product usage and so on. All these tools must be visible to all people in the team, specially if they work distributed across different locations.

The current solution

Today, the solution is to set up a TV at the end of the desks and open multiple web browser windows displaying all the tools the team uses. Something like the following:

Team stats

Team stats is an Electron application that tries to solve this problem a little better. There are plenty of feature ideas that could be implemented to be a better solution than the browser, helping teams communicate better through their TVs. Team Stats is an attempt to implement them.

Things like:

  • Tabs configured from JSON file
  • Automatic zoom and scrolling configurations
  • Automatic and individual page refreshing
  • Better use of the screen space
